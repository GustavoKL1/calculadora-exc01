# Calculadora

Uma calculadora simples feita com React + TypeScript + Vite.

![React](https://img.shields.io/badge/React-19-61DAFB)
![TypeScript](https://img.shields.io/badge/TypeScript-6-3178C6)
![Vite](https://img.shields.io/badge/Vite-8-646CFF)

## Como rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Operacoes basicas: +, -, *, /
- Potencia (^) e raiz quadrada (√)
- Tema escuro
- Teclado numerico

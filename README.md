## ✨ Features

| Operação | Botão |
|----------|-------|
| Soma | `+` |
| Subtração | `−` |
| Multiplicação | `*` |
| Divisão | `/` |
| Potência | `^` |
| Raiz quadrada | `√` |
| Limpar | `C` |

- ✅ Tema escuro elegante
- ✅ Suporte a teclado numérico
- ✅ Expressão visível enquanto digita
- ✅ Design responsivo

---

## 🚀 Como Rodar

```bash
# Instalar dependências
npm install

# Servidor de desenvolvimento
npm run dev

# Build de produção
npm run build

# Preview do build
npm run preview
```
